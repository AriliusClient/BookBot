# BookBot
A bot that writes books.

## Usage
Prefix: `@bb`<br>
To view all commands, run `@bb help`

## Commands
### AsciiBook
Generates a book with random ascii characters
#### Arguments
None

#### Example
- `@bb asciibook`

### BinaryBook
Generates a book with random binary characters
#### Arguments
1. (Optional) boundary (Integer)

#### Example
- `@bb binarybook`
- `@bb binarybook 2000`

### Help
Shows all commands
#### Arguments
None

#### Example
`@bb help`

### Test
Tests if the mod functions correctly
#### Arguments
None

#### Example
`@bb test`
